<Update date="2026-05-14" title="Introduce Prompty tracing and Azure OpenAI tracing infrastructure">
Adds tracing infrastructure across the Contoso chat API. Introduces PromptyTracer and console tracer integrations, wires Tracer at startup, and exposes init_tracing to switch between local tracing (PromptyTracer) and remote tracing (OpenTelemetry with Azure Monitor). Updates various modules (chat_request, product, evaluators, tracing) to decorate functions with @trace and to load configuration from dotenv. Enables embeddings generation and semantic product retrieval via Azure OpenAI and Azure Cognitive Search, enabling end-to-end tracing for requests.
